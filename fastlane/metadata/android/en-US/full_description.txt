<p><i>Screenlite Web Kiosk</i> is a simple Android kiosk browser app that displays web content in full-screen mode.</p><p><br><b>Key Features</b></p><ul><li>Displays any website in immersive full-screen mode</li><li>Automatically launches on device boot</li><li>Brings itself back to the foreground if minimized</li><li>Click and scroll are disabled in WebView</li><li>Supports website rotation (0°, 90°, 180°, 270°), even if the device doesn’t support orientation changes</li><li>Automatic retry mechanism with exponential backoff for network failures</li><li>Automatically injects responsive viewport meta tags</li><li>Special UI adaptations for Android TV devices</li><li>Hides system bars and navigation elements</li><li>Prevents device from going to sleep</li><li>Real-time network connectivity monitoring</li><li>Error handling with user-friendly messages</li><li>Compatible with both Android TV and mobile devices</li></ul><p><b>Usage Instructions</b></p><ol><li>Install the APK on your Android device.</li><li>Grant the Display over other apps permission to Screenlite Web Kiosk.</li><li>Launch the app.</li><li>Access settings by tapping the screen 5 times quickly (or pressing the center button on a TV remote).</li><li>Enter the website URL you want to display.</li><li>Set the app visibility check interval (recommended 5–10 seconds). If you need time to configure the device, set a higher number.<br><i>Note: The app won’t automatically bring itself back to the front when this timer ends.</i></li><li>Choose screen rotation if needed.</li><li>Save your settings; the app will restart with the new configuration.</li></ol>
